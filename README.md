# Description

[Nest](https://github.com/nestjs/nest) is a TypeScript framework starter repository.

## Installation

```
pnpm install
```
## Running the app
# Start
```
pnpm run start
```
# Watch mode
```
pnpm run start:dev
```
# Production mode
```
pnpm run start:prod
```
## Test
# Unit tests
```
pnpm run test
```

# End-to-end tests
```
pnpm run test:e2e
```

# Test coverage
```
pnpm run test:cov
```
